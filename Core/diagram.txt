

* Game
    - init/destroy SDL
    - dispatch SDL input events to gameState_
    - perform game loop, keep track of elapsed time
    - call gameState_.update() and gameState_.render()

    * GameState
        - has Entity list
        - has System list


* Entity
    - Transform Component
    - List of components

* Component
    - Entity reference
    - Component data
    - Useful references to other components

* System
